## 时间日期类指令

- 基本语法

    date  （功能描述：显示当前时间）
    date + %Y  （功能描述：显示当前年份）
    date + %m  （功能描述：显示当前月份）
    date + %d  （功能描述：显示当前是哪一天）
    date "+%Y-%m-%d %H:%M:%S"   （功能描述：显示年月日时分秒, - 可用空格代替）

- 应用实例

### 设置日期

- 基本语法

  date -s "字符串时间"

### cal指令

查看日期指令

- 基本语法
  
  cal[选项] (功能描述：不加选项，显示本月日历)

  cal  2020   (显示2020年日历)
